- 토큰 생성
오른쪽 프로필 클릭 - Settings - Developer settings - Personal access tokens - Generate new token

# 6. git 명령어 정리

- dir		// 디렉토리 내 폴더 및 파일 확인
- git init		// git 저장소 생성
- git config --global user.email "yangjh0130@naver.com"		// 사용자 이메일 설정
- git config --global user.name "yangjh0130"			// 사용자 이름 설정
- git add blockchain_0530.txt					// 파일 추적 시작
- git commit -m blockchain_0530.txt				// 코드 변화 기록
- git log							// commit된 스냅샷 표시
- git status 							// git 상태 확인

- git remote add origin https://github.com/yangjh0130/blockchain_woosuk	// 원격 저장소 연결
- git branch -M main						// 기본 브랜치를 main으로 변경
- git push -u origin main					//  로컬 저장소의 main 브랜치를 자동으로 origin 원격저장소의 main 브랜치와 연결
  명령어 입력 시 github sign in 화면이 나오면 Token 클릭 후 발급받은 토큰 키 입력
- git add blockchain_0530.txt					// 수정한 파일 add
- git commit -m "0530 정리"					// 0530 정리 이름으로 commit
- git push							// 원격 저장소로 전송

- init -> 사용자 설정 -> add -> commit -> 원격 저장소 연결 -> 브랜치 설정 -> push
- 파일 수정 시 add -> commit -> push
- log와 status로 확인
